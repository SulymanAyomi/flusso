# **Flusso**

_From prompt to project in seconds._

Flusso is an AI-powered productivity and project management app that transforms a simple prompt into a fully structured project — complete with tasks, timelines, and priorities. Designed for individuals and teams, it organizes work into workspaces, manages tasks with smart scheduling and prioritization, and enables seamless collaboration — all while AI handles the heavy lifting of planning.

## 🚀 Features

- **AI Project Creation** – Generate a complete project from a single prompt.
- **Manual Project Creation** – Build projects from scratch with your own structure.
- **Task Management** – Add, edit, and break down tasks into subtasks.
- **Comments & Activity Tracking** – Keep discussions and progress history within tasks.
- **Tags** – Organize projects and tasks with tags for easy filtering.
- **Multiple Views** – Switch between Table, Board, and Calendar views.
- **Dashboard Analytics** – Track progress and performance across workspaces.
- **My Tasks** – See all your assigned tasks, segmented by Today, Upcoming, and Overdue.

## 📦 Installation & Setup

### **Prerequisites**

- [Node.js](https://nodejs.org/) (v16+ recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)
- A [Flusso API key](https://flussoapp.com/) for AI-powered features

### **Steps**

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/flusso.git
   cd flusso
   ```
2. **Install dependencies**
   ```bash
   npm install
   ```
   or
   ```bash
   yan install
   ```
3. **Set up environment variables**
   ```bash
   API_KEY=your_flusso_api_key
   DATABASE_URL=your_database_url
   ```
4. **Run the development server**
   ```bash
   npm run dev
   ```
   or
   ```bash
   yarn dev
   ```

## 🛠 Usage

### 1. Creating a Project with AI

1. Log in to your account.
2. Click **Create Project** → **From Prompt**.
3. Enter your idea (e.g., _Build a personal finance tracking app_).
4. Flusso will generate:
   - Project name & description
   - Structured tasks with due dates
   - Priorities and dependencies

---

### 2. Creating a Project Manually

1. Click **Create Project** → **Manual**.
2. Enter project name, description, and tags.
3. Add tasks manually or break them into subtasks.

---

### 3. Managing Tasks

- View tasks in **Table**, **Board**, or **Calendar** view.
- Filter and sort by status, due date, priority, or tags.
- Add comments, track activity, and assign tasks to teammates.

---

### 4. Tracking Progress

1. Use **Dashboard Analytics** for workspace and project insights.
2. Go to **My Tasks** to see all your assignments segmented by:
   - **Today**
   - **Upcoming**
   - **Overdue**

## 📖 Getting Started

### 1. Sign Up

- Go to the Flusso web app.
- Click **Sign Up** and create an account using your **email** and **password**.

### 2. Log In

- Enter your credentials on the **Login** page.

### 3. Create Your First Workspace

- Upon first login, you’ll be prompted to create a **workspace**.
- Enter a name and confirm.

### 4. Explore Your Dashboard

- View workspace statistics, navigate to projects, and check **My Tasks**.

### 5. Create Your First Project

- Choose **From Prompt (AI)** or **Manual**.

### 6. Manage Tasks

- Add/edit tasks, subtasks, and comments, and track activity.
